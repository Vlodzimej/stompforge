# StompForge 0.0.1-alpha

Первый публичный alpha-выпуск модульного гитарного процессора StompForge.

## Основное

- Windows x64 VST3 и Standalone.
- Матрица педалборда до 4x3 с drag-and-drop и пустыми ячейками.
- Одиннадцать типов модулей, включая два amp, IR player, тюнер и NAM MODELER.
- До трёх основных регуляторов в карточке и расширенные параметры в `MORE`.
- Вертикальные Input/Output фейдеры с индикаторами уровня.
- Сохранение цепочки и параметров в сессии DAW.

## MODELER

MODELER использует NeuralAmpModelerCore v0.5.4, загружает mono-in/mono-out
`.nam`, согласует sample rate с хостом и использует независимое состояние для
двух stereo-каналов. Модели в пакет не включены.

## Известные ограничения

- Это тестовый alpha-выпуск, не предназначенный для критичных live-сценариев.
- `.nam` и IR остаются внешними файлами и должны сохраняться пользователем.
- Загрузка крупной `.nam` может кратковременно задержать UI.
- Проверенная release-платформа — Windows 10/11 x64.
- Бинарники пока не подписаны цифровой подписью.
